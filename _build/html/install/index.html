Installation

Free tier: public cloud¶

Run on Nvidia GPUs for free! Just register (allow 1 business day for processing) and install our python frontend client. Unlike before, no need to install the backend. See our simulation tutorials afterwards. Community cloud only supports simulations (not inverse design). Everything uploaded is publicly visible. Simply do:

pip install luminescent

Some systems can take few minutes to initialize dependencies on first run.

Enterprise tier¶

Private cloud¶

Similar to community cloud but private and secure plus superior GPUs. $3 GPU hour for latest Blackwell class Nvidia GPUs, cheaper than from AWS or GCP.

Local install¶

Pay once and run forever! Backend binaries for running on your own machine. Any private cloud spend counts as credit towards local install purchase. Vice versa.
